# Import Architecture dans Lucidchart

## Méthodes d'import disponibles

### 1. Diagramme ER (Entity Relationship)
- Fichier : `erd_import.csv`
- Dans Lucidchart : **File > Import > CSV**
- Choisir "Entity Relationship Diagram"

### 2. Architecture en couches
- Fichier : `architecture_layers.csv`
- Dans Lucidchart : **File > Import > CSV**

### 3. Diagramme de flux utilisateur
- Fichier : `user_flow.csv`
- Dans Lucidchart : **File > Import > CSV**

### 4. Mermaid (alternative)
- Fichier : `mermaid_diagrams.md`
- Dans Lucidchart : Insérer un bloc "Mermaid" et coller le code

## Instructions Lucidchart

1. Ouvrir Lucidchart
2. Créer un nouveau document
3. File > Import Data > CSV
4. Sélectionner le fichier CSV souhaité
5. Mapper les colonnes selon les suggestions
6. Ajuster le layout automatiquement généré
